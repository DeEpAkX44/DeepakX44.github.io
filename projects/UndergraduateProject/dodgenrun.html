Dodge N Run

Project information

  • Category: Undergraduate Project (Unity)
  • Project date: July, 2022
  • Project URL: DodgeNRun

Developed a Fall Guys-inspired multiplayer party platformer where 5+ players compete across obstacle stages. Implemented real-time networking with Photon Unity to enable competitive gameplay. Built a versatile kinematic character controller for both Player and AI characters. Designed the controller to support multiple genres including platformer, FPS, third-person, and adventure.
